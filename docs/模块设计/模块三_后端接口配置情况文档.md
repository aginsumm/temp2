## 二、核心接口详情

### 2.1 实体管理（4 个）

| 接口                                   | 方法   | 功能         | 说明              |
| -------------------------------------- | ------ | ------------ | ----------------- |
| `/api/v1/knowledge/entity`             | POST   | 创建实体     | 支持 7 种实体类型 |
| `/api/v1/knowledge/entity/{entity_id}` | GET    | 获取实体详情 | 返回完整实体信息  |
| `/api/v1/knowledge/entity/{entity_id}` | PUT    | 更新实体     | 更新实体属性      |
| `/api/v1/knowledge/entity/{entity_id}` | DELETE | 删除实体     | 级联删除关系      |

### 2.2 关系管理（4 个）

| 接口                                               | 方法   | 功能         | 说明               |
| -------------------------------------------------- | ------ | ------------ | ------------------ |
| `/api/v1/knowledge/relationship`                   | POST   | 创建关系     | 支持 10 种关系类型 |
| `/api/v1/knowledge/relationship/{relationship_id}` | GET    | 获取关系详情 | 返回完整关系信息   |
| `/api/v1/knowledge/relationship/{relationship_id}` | PUT    | 更新关系     | 更新关系属性       |
| `/api/v1/knowledge/relationship/{relationship_id}` | DELETE | 删除关系     | 删除指定关系       |

### 2.3 搜索和检索（4 个）

| 接口                                             | 方法 | 功能               | 说明                          |
| ------------------------------------------------ | ---- | ------------------ | ----------------------------- |
| `/api/v1/knowledge/search`                       | POST | 搜索实体           | 关键词搜索，支持分页          |
| `/api/v1/knowledge/search/advanced`              | POST | 高级搜索           | 多条件组合搜索                |
| `/api/v1/knowledge/graph`                        | GET  | 获取图谱数据       | 返回 nodes、edges、categories |
| `/api/v1/knowledge/entity/{entity_id}/detail`    | GET  | 获取实体详情及关联 | 返回实体、关系、相关实体      |
| `/api/v1/knowledge/entity/{entity_id}/relations` | GET  | 获取实体的所有关系 | 返回关系列表                  |

### 2.4 统计和分析（6 个）

| 接口                                                | 方法 | 功能             | 说明                 |
| --------------------------------------------------- | ---- | ---------------- | -------------------- |
| `/api/v1/knowledge/stats`                           | GET  | 获取图谱统计数据 | 返回实体数、关系数等 |
| `/api/v1/knowledge/categories`                      | GET  | 获取实体类型列表 | 返回 7 种类型配置    |
| `/api/v1/knowledge/regions`                         | GET  | 获取地域列表     | 返回所有地域         |
| `/api/v1/knowledge/periods`                         | GET  | 获取时期列表     | 返回所有时期         |
| `/api/v1/knowledge/analysis/connectivity`           | GET  | 分析图谱连通性   | 返回连通分量信息     |
| `/api/v1/knowledge/analysis/centrality/{entity_id}` | GET  | 分析实体中心性   | 返回中心性指标       |
| `/api/v1/knowledge/analysis/community`              | GET  | 分析社区结构     | 返回社区划分结果     |

### 2.5 用户交互（7 个）

| 接口                                            | 方法   | 功能               | 说明               |
| ----------------------------------------------- | ------ | ------------------ | ------------------ |
| `/api/v1/knowledge/favorite`                    | POST   | 添加收藏           | 收藏实体           |
| `/api/v1/knowledge/favorite`                    | DELETE | 取消收藏           | 取消实体收藏       |

## 三、数据模型

### 3.1 核心模型

**Entity（实体）**

- id, name, type, description, region, period, coordinates, meta_data, importance, created_at, updated_at
- type: inheritor | technique | work | pattern | region | period | material

**Relationship（关系）**

- id, source_id, target_id, relation_type, weight, meta_data, created_at
- relation_type: inherits | origin | creates | flourished_in | located_in | uses_material | has_pattern | related_to | influenced_by | contains

**Favorite（收藏）**

- id, user_id, entity_id, created_at

**Feedback（反馈）**

- id, user_id, entity_id, feedback_type, content, rating, created_at

### 3.2 辅助模型

- SearchHistory（搜索历史）
- GraphData（图谱数据）：nodes, edges, categories
- GraphNode（图节点）：id, name, category, symbolSize, value, itemStyle
- GraphEdge（图边）：source, target, relationType, lineStyle

## 七、部署说明

## 八、已知限制

1. **数据规模限制**：当前使用 SQLite，适合中小规模数据（<10 万实体）
2. **搜索功能限制**：仅支持关键词搜索，不支持语义搜索
3. **图谱渲染限制**：大规模图谱（>1000 节点）可能存在性能问题
4. **内存存储**：部分数据使用内存存储，重启后可能丢失
5. **图数据库缺失**：未使用 Neo4j 等图数据库，复杂查询性能受限
